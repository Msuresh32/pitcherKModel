Analysis package for Mani review
================================

This folder contains the copied analysis outputs for the pitcher-K head-to-head review.

Folders
-------

- outputs/mani_regrades: Mani 2025+2026 regrades against the 3-book consensus sharp close, plus DK-drift exhibits.
- outputs/band_analysis: Edge-band analysis, including the 15%+ band test.
- outputs/execution_capture: Lekobe Kalshi execution-capture outputs for the 13 placed bets.
- outputs/calibration: 2025 book sharpness and calibration outputs used to choose the Pin+FD+BetOnline benchmark.
- outputs/fade_rule_scale: Mechanical fade-favored-overs scale backtest outputs.
- scripts: Snapshots of reusable lekobe scripts involved in sharp benchmarking, grading, scanning, and execution analysis.

Benchmark
---------

Sharp close means Pinnacle + FanDuel + BetOnline, de-vigged per book first, then averaged. Two-book and one-book fallbacks are flagged in the output files where applicable.
